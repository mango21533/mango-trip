Mango · Travel · v2.0 Generator

🥭 这是什么

任意城市 × 任意天数 PWA 行程生成器。输入城市名 + 天数 + 出发日期，自动搜索高德 POI，智能拼接美食 + 景点 + 住宿 + 路线，生成专属你的旅行手册。

⚙️ 工作原理

  • POI 搜索 — 高德地图 API 实时查景点/餐厅/酒店
  • 路径规划 — 高德驾车 API 算距离 + 时间 + 费用
  • 数据缓存 — localStorage 存 7 天，二次访问秒开
  • 离线可用 — Service Worker 缓存所有资源

📱 使用说明

  • 点 ✓ 打卡 — 标记已去，自动存本地
  • 点 🗺️ 导航 — 一键跳转高德地图
  • 切换城市 — 顶部表单填新城市，点生成
  • 添加到主屏 — 像 App 一样用，离线可用
